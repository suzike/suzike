## Overview

I focus on automotive thermal management application software and AI-native engineering platforms.

My daily work is centered on MATLAB, Simulink, model-based development, control logic, software requirements, detailed design, MIL/HIL testing, and engineering documentation. I am also building toward an enterprise AI Native R&D platform that connects requirements, architecture, modeling, testing, calibration, delivery, and knowledge accumulation.
